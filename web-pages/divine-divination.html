divine divination

date: april 2023

tools: html, css, js [vanilla + rita.js]

OVERVIEW

what is it?

a virtual divination ritual utilizing the concept of bloodletting as a means of divination

intent

to reimagine the physicality of divination rituals in a web based platform

DESIGN



for the design/concept, i derived the overall idea of utilizing blood as an element after donating blood. stemming off this experience, i thought back to the ancient ritual of bloodletting, and while it was not explicitly used for divination purposes, i wanted to spin the practice in a divination lens. the fortune part, i wanted to utilize the rita.js haiku generator, since it produces cryptic messages made possible by the randomness of rita.js.

BUILD

REFLECTION

design

creating the assets on procreate were probably the most time consuming part of the design process! however, i really wanted to use my own assets for this project and am glad with how it turned out! as for the figma design, i had a clear vision of how i wanted the “ritual” to go, and so i did not encounter much road blocks in the creative process. 


build

i had a few hiccups along the way of building the site from start to end! integrating the rita.js library was a bit of a pain, but after some troubleshooting i was able to make it work to a certain extent... also, figuring out the timing for the css animtions took some trial and error, but i think i have reached a good conclusion. i was inspired by this codepen for the scene with falling text and proud of finding a way to integrate it into my project in my own style! i also got a lot of practice with css animations and so that was quite a lovely time!


conclusion

overall, i am glad with how the ritual turned out and i learned quite a lot along the way! sometimes, i like to revisit the site for days where im not sure where life is taking me hehe
